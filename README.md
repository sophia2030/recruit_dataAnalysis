## 📌 项目简介

针对游戏行业数据分析岗位信息分散、求职决策缺乏数据支撑的痛点，设计并实现了一套从**数据采集**到**智能决策**的全链路情报分析系统。

**在线演示**：[http://167.179.77.108:8501/](http://167.179.77.108:8501/)

### 核心能力

| 模块           | 功能                                                         |
| :------------- | :----------------------------------------------------------- |
| 🕷️ **数据采集** | 逆向分析企业级 SaaS 平台 API，支持多公司并发爬取、断点续传、自动重试 |
| 🧹 **数据清洗** | 基于 Pandas 构建清洗流水线，提取薪资、经验、学历、技能等结构化特征 |
| 📊 **技能分析** | 黑白名单词库过滤 + 熟练度量化 + 技能重要程度排序             |
| 🎯 **简历匹配** | 基于 Tfidf + 余弦相似度，上传简历即可获得 Top 10 匹配岗位推荐 |
| 📈 **能力画像** | 五维能力模型（技术/建模/商业/可视化/工程）+ 双雷达图差距诊断 |
| 🖥️ **交互看板** | Streamlit + Plotly 搭建，支持多维度筛选与交互式探索          |

## 🎯 项目背景

在求职数据分析岗位的过程中，我发现招聘信息分散在各平台，且岗位描述中的技能需求、薪资水平、经验要求等信息难以系统化对比。为了解决 **“投哪家、学什么、差在哪”** 三个核心问题，我构建了这套情报分析系统。

## 🏗️ 技术架构

```
技术架构（分层说明）

1. 数据采集层
   - 多公司爬虫（莉莉丝/米哈游/鹰角/叠纸）
   - 断点续传（progress.txt 记录进度）
   - 自动重试（单页最多 3 次）
   - 随机延时（1.5~3.5 秒）
   └── 输出：原始 JSON 数据

2. 数据清洗与特征工程
   - Pandas 清洗流水线
     - 薪资中值提取（如 20-40K → 30K）
     - 经验年限提取（如 3-5年 → 4年）
     - 技能词库匹配（白名单 + 黑名单过滤）
     - 熟练度量化（精通/熟练掌握/熟练/熟悉/了解/掌握）
   └── 输出：结构化 CSV 数据

3. 建模与决策引擎
   - 简历匹配引擎（Tfidf + 余弦相似度）
   - 五维能力模型（技术/建模/商业/可视化/工程）
   - 双雷达图 + 差距分析（优势维度 / 盲区识别）
   └── 输出：匹配度排名 + 学习建议

4. 可视化与部署
   - Streamlit 交互式看板
   - Plotly 动态图表
   - VPS 公网部署（systemd 保活）
   └── 输出：公网可访问看板
```

## 🚀 快速开始

### 环境要求

- Python 3.9+
- pip / conda

### 安装依赖

```bash
# 克隆项目
git clone https://github.com/yourusername/recruit_dataAnalysis.git
cd recruit_dataAnalysis

# 安装依赖
pip install -r requirements.txt
```

### 运行爬虫

```bash
# 进入爬虫目录
cd crawler

# 修改 DPID_LIST（在代码顶部用户配置区）
# 运行爬虫
python crawler.py
```

### 启动看板

```bash
# 返回项目根目录
cd ..

# 运行 Streamlit 看板
streamlit run dashboard.py
```

浏览器将自动打开 `http://localhost:8501`

## 🔧 功能模块详解

### 1. 数据采集模块

- 逆向分析 `xiaoke.cn` API 接口，定位 `detailModule` 数据端点
- 支持通过 `DPID_LIST` 配置多公司批量爬取
- 断点续传：通过 `progress.txt` 记录进度，中断后可继续
- 自动重试：单页失败最多重试 3 次
- 随机延时：1.5~3.5 秒间隔，降低反爬风险
- 时间戳版本命名：每次运行生成独立 JSON 文件，避免数据覆盖

### 2. 数据清洗与分析模块

- 薪资中值提取：从 `"20-40K·16薪"` 中提取中位数
- 经验年限提取：从 `"3-5年"` 中提取数值范围
- 技能词库：支持从 `wordlists/skills/` 加载自定义白名单
- 停用词库：支持从 `wordlists/stopwords/` 加载自定义黑名单
- 黑白名单过滤：白名单词强制保留，黑名单词仅当不在白名单时删除
- 熟练度量化：识别 `精通/熟练掌握/熟练/熟悉/了解/掌握` 并加权

### 3. 简历匹配引擎

- 基于 `TfidfVectorizer` 将岗位描述向量化
- 基于余弦相似度计算简历与各岗位的匹配度
- 输出 Top 10 匹配岗位及匹配度百分比
- 自动提取最匹配岗位的高频技能作为学习建议

### 4. 能力画像与差距分析

- **五维能力模型**：技术硬实力 · 数据科学/建模 · 商业/游戏理解 · 可视化/沟通 · 工程/工具
- **双雷达图**：对比“市场需求平均”与“我的能力”
- **差距分析**：自动识别优势维度（超出市场）与盲区（落后市场）
- **学习建议**：针对薄弱维度生成具体行动建议

### 5. 交互式看板

| 模块       | 说明                                   |
| :--------- | :------------------------------------- |
| KPI 卡片   | 岗位总数、薪资中位数、平均经验、公司数 |
| 薪资分布   | 直方图展示薪资区间分布                 |
| 公司对比   | 各公司薪资中位数对比柱状图             |
| 高频技能   | Top 15 技能词频排名                    |
| 城市分布   | Top 8 招聘城市饼图                     |
| 学历要求   | 学历分布条形图                         |
| 招聘趋势   | 月度岗位数量折线图（按年分界）         |
| 简历匹配   | 上传简历，输出匹配度 Top 10            |
| 能力雷达图 | 双雷达图 + 差距分析列表                |
